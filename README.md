# Optimization Algorithms

This repository contains codes, algorithm implementations, and lecture notes developed within the scope of the **Optimization Algorithms** course.

## About

This project includes applications of various heuristic and mathematical optimization methods implemented using MATLAB and Python languages. The aim is to reinforce the theoretically learned algorithms with coding practices.

### Topics and Algorithms Included
*   Artificial Immune Systems (CLONALG)
*   Genetic Algorithms
*   Steepest Descent Method
*   Other constrained and unconstrained optimization techniques
